# TodaTrack

Steps:
1. Download all resouces in (OOP folder) or (TodaTrackl.zip)
2. If you Downloaded the (TodaTrackl.zip) unzip it
3. Open your compiler (Ex. VS Code)
4. Make sure that your compiler have java extentions
5. If you don't have java extentions. Install (Extension Pack for Java)
6. Make sure and double check if all resources are complete
7. If you were using VS Code go to "File"
8. Next is go to "Open Folder" or "Ctrl+k+Ctrl+O"
9. Select the OOP Folder or the resources you've downloaded and Select it
10. Open the "TodaTrackApp.java" and run it
11. After all you can now explore the system "TodaTrack"
